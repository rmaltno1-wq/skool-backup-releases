# Skool Scraper — Premium Course & Video Backup Utility

Skool Scraper is a local desktop application designed to help you back up and archive your subscribed Skool classroom courses, high-definition videos, and attachments locally. Learn offline, preserve your course materials, and keep your files secure.

🌐 **Official Website**: [skoolscraper.sillgarden.app](https://skoolscraper.sillgarden.app)

## 🚀 Key Features

*   **HD Video Extractor**: Intercepts native HLS streams (including Fragmented MP4s) and automatically merges separate video/audio tracks into high-quality, standard `.mp4` video files.
*   **Smart Caching & Resume**: Automatically skips already-downloaded lessons and resumes failed runs from where they stopped, saving time and bandwidth.
*   **Offline Markdown & HTML Archive**: Compiles lesson texts, bodies, and resource links into beautiful offline HTML files and clean Markdown documents.
*   **Secure Local Auth**: Connects directly to Skool via an automated local browser. Your credentials and cookies are stored 100% locally on your computer.
*   **Attachment Downloader**: Scrapes and downloads PDFs, ZIPs, sheets, documents, and assets directly to categorized local folders.

## 🛠️ Installation Guide

### macOS
1. Download the **`Skool-Scraper-Mac.dmg`** installer.
2. Double-click the `.dmg` file to mount it.
3. Drag the **SkoolScraper** icon into your **Applications** folder.
4. Open your Applications folder and launch the app. 
   *(Note: Because this is a self-signed binary, you may need to right-click -> Open, or go to **System Settings -> Privacy & Security** and click **Open Anyway** if macOS Gatekeeper flags it).*

### Windows
